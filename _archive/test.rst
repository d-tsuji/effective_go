テスト
============================================

公式ドキュメントはこちら。go test コマンドによるテストがされることを想定して作成されている。

https://golang.org/pkg/testing/

--------------------------------------------
規則
--------------------------------------------

- テストしたい関数は大文字から始まる関数とする
- テスト関数内では Error を用いる
- テスト関数が含まれるファイル名のsuffixは _test.go とする

テスト関数の例は以下

.. code-block:: go

    func TestAbs(t *testing.T) {
        got := Abs(-1)
        if got != 1 {
            t.Errorf("Abs(-1) = %d; want 1", got)
        }
    }
